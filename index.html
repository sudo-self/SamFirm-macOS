This guide shows how to check for the latest firmware, download it, and decrypt it using samloader.

  • Python 3.10 or 3.11 (‚ùó Avoid Python 3.13 ‚Äî causes runtime errors)
  • Install samloader:

  • macOS, Linux, or Windows with terminal access
  • Samsung model: SM-S921U1 (Galaxy S24 Ultra - US Unlocked)
  • CSC (region): XAA (Unbranded US)

üîç Step 1: Check for Latest Firmware

samloader -m SM-S921U1 -r XAA checkupdate

Expected output:

S921U1UES4BYE4/S921U1OYM4BYE4/S921U1UES4BYE4/S921U1UES4BYE4

This confirms the firmware is available.

‚¨áÔ∏è Step 2: Download the Firmware

samloader -m SM-S921U1 -r XAA download -v S921U1UES4BYE4 -O ./firmware
  • -m: device model
  • -r: region code
  • -v: full firmware version
  • -O: output directory for encrypted firmware

üîì Step 3: Decrypt the Firmware

After the download completes:

samloader -m SM-S921U1 -r XAA decrypt -v S921U1UES4BYE4 -V 4 -i ./firmware/SM-S921U1_*.zip -o ./firmware/decrypted
  • -V 4: encryption version (usually 4)
  • -i: input ZIP file path
  • -o: output directory for decrypted Odin files

üßº Clean Up

rm ./firmware/SM-S921U1_*.zip

üêõ Troubleshooting

  • No latest firmware available: Try different model or CSC (e.g., SM-S921U1 + XAA)
  • IndexError / string index out of range: Downgrade to Python 3.10 or 3.11
  • Decrypt can't find ZIP: Ensure file exists and -i path is correct
